Какие ограничения на хостинге Евробайт?
===============================================
В зависимости от тарифа, не допускается пиковое использование процессами пользователя более чем 100% ресурсов одного ядра процессора.

Уровень допустимой нагрузки для стандартных тарифов виртуального хостинга:
3Гб — 15%
5Гб — 22.5%
8Гб — 35%
15Гб — 50%


Уровень допустимой нагрузки для тарифов CMS-хостинга: 
8Гб — 25%
15Гб — 50%
30Гб — 100%


Уровень допустимой нагрузки для VIP-хостинга: 100%

 

Для PHP скриптов действуют следующие ограничения:

Максимальный размер выделяемой памяти (memory_limit) - 400 Мb
Максимальное время выполнения PHP скрипта (max_execution_time) - 60 секунд.
Максимальное количество одновременных подключений пользователя баз данных к серверу баз данных (max_user_connections) - 30.
Максимальный объем загружаемого файла через скрипты (upload_max_filesize) - 250 Мb

CGI лимиты web-сервера:

40 процессорных секунд может использовать один cgi процеcc
Разрешается использовать одним cgi процессом 200 Мb памяти
Один CGI процесс может запустить 30 дочерних процессов

Лимиты операционной системы:

Максимальное количество одновременных http соединений - 40 для тарифов виртуального хостинга, 80 - для тарифов VIP и CMS хостинга
Максимальное количество процессов, которые может одновременно запустить пользователь - 30
Максимальное количество памяти, которое может использовать в скриптах пользователь - 300 Мb
Максимальное количество файлов, которые одновременно может открыть пользователь - 3176
Максимальный размер файла, который может создать пользователь - 3 Gb
Максимальное количество открытых сессий пользователя - 5

Почтовые лимиты:

Максимальное количество писем отправляемых через наш smtp сервер - не более 400 писем в час
Максимальный размер одного письма - 50 Мb
Максимально количество получателей (поля TO, CC, BCC) - 100
